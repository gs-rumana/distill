# Distill - Get Tasks done with Pomodoro

This app is a productivity app that pick top tasks for users. Users can add tasks, run pomodoro for any task, and see the stats.

## Tech Stack

- React Native
- Expo
- TypeScript

## Key Features

- Tasks CRUD Operations
- Top tasks on home screen
- Pomodoro timers
- Analytics

## Things to note

- Use colors from app theme
- Useful constants are available
- Uses `expo-router` for navigation
- Type definitions are available for everything
- Take a look at `components` folder
- Use `npx expo` to install any package
